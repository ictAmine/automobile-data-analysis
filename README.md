# Automobile Data Analysis

## Objective
Explore and analyze automobile data to understand which features
influence car prices.

## Dataset
- automobile_dataset.csv
- Contains technical, physical and categorical attributes of cars
- Includes missing values and mixed data types

## Steps performed
1. Data loading and inspection
2. Data cleaning (missing values, type correction)
3. Exploratory Data Analysis (EDA)
   - Descriptive statistics
   - Skewness analysis
   - Correlation analysis

## Notes
- Dataset is small, conclusions are indicative
- Median preferred over mean for skewed distributions
